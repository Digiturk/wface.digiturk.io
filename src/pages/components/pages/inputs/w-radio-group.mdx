# WRadioGroup

WRadioGroup @material-ui RadioGroup'dan türetilmiştir.
RadioGroup'un propertylerine erişmek için [tıklayın](https://material-ui.com/api/radio-group/).

### Ek Özellikler

Eklenen ve değiştirilen özellikler aşağıdaki gibidir:

| Adı                 | Veri Tipi                                                     | Default | Açıklama                                        |
|:--------------------|:--------------------------------------------------------------|:--------|:------------------------------------------------|
| label               | string                                                        |         | Radio button için label                         |


### Örnekler

<LiveCode title="Temel Kullanım" code={`
<WFace.WRadioGroup value="2">
  <WFace.WRadio value="1" label="Radio 1"/>
  <WFace.WRadio value="2" label="Radio 2"/> 
</WFace.WRadioGroup>  
`}/>
