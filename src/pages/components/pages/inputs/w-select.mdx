# WSelect

WSelect material-ui-pickers react-select'den türetilmiştir.
react-select'un propertylerine erişmek için [tıklayın](https://react-select.com/home).

### Ek Özellikler

Eklenen ve değiştirilen özellikler aşağıdaki gibidir:

| Adı                 | Veri Tipi                                                     | Default | Açıklama                                        |
|:--------------------|:--------------------------------------------------------------|:--------|:------------------------------------------------|
|                     |                                                               |         |                                                 |
