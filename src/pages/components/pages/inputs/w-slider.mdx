# WSlider

WSlider @material-ui Slider'dan türetilmiştir.
Slider'un propertylerine erişmek için [tıklayın](https://material-ui.com/api/slider/).

### Ek Özellikler

Eklenen ve değiştirilen özellikler aşağıdaki gibidir:

| Adı                 | Veri Tipi                                                     | Default | Açıklama                                        |
|:--------------------|:--------------------------------------------------------------|:--------|:------------------------------------------------|
|                     |                                                               |         |                                                 |
