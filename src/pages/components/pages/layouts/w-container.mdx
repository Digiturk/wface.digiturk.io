# WContainer

WContainer @material-ui Container'dan türetilmiştir.
Container'un propertylerine erişmek için [tıklayın](https://material-ui.com/api/container/).

### Ek Özellikler

Eklenen ve değiştirilen özellikler aşağıdaki gibidir:

| Adı                 | Veri Tipi                                                     | Default | Açıklama                                        |
|:--------------------|:--------------------------------------------------------------|:--------|:------------------------------------------------|
|                     |                                                               |         |                                                 |


### Örnekler

`}/><br/>
