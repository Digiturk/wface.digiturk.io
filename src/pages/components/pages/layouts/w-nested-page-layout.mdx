# WNestedPageLayout

WNestedPageLayout birden fazla sayfa hiyerarşik bir şekilde yönetilebilir. 

### Ek Özellikler

Eklenen ve değiştirilen özellikler aşağıdaki gibidir:

| Adı                 | Veri Tipi                                                     | Default | Açıklama                                          |
|:--------------------|:--------------------------------------------------------------|:--------|:--------------------------------------------------|
| root                | WNestedPageItem                                               |         | Ana bileşen                                       |
| state               | WNestedPageLayoutState                                        |         | Bileşenin durumu                                  |
| onStateChanged      | (state: WNestedPageLayoutState) => void;                      |         | Bileşenin durum değişikliklerini veren fonksiyon  |
