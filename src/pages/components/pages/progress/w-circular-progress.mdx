# WCircularProgress

WCircularProgress @material-ui CircularProgress'dan türetilmiştir.
CircularProgress'un propertylerine erişmek için [tıklayın](https://material-ui.com/api/circular-progress/).

### Ek Özellikler

Eklenen ve değiştirilen özellikler aşağıdaki gibidir:

| Adı                 | Veri Tipi                                                     | Default | Açıklama                                        |
|:--------------------|:--------------------------------------------------------------|:--------|:------------------------------------------------|
|                     |                                                               |         |                                                 |
