# WLinearProgress

WLinearProgress @material-ui LinearProgress'dan türetilmiştir.
LinearProgress'un propertylerine erişmek için [tıklayın](https://material-ui.com/api/linear-progress/).

### Ek Özellikler

Eklenen ve değiştirilen özellikler aşağıdaki gibidir:

| Adı                 | Veri Tipi                                                     | Default | Açıklama                                        |
|:--------------------|:--------------------------------------------------------------|:--------|:------------------------------------------------|
|                     |                                                               |         |                                                 |
