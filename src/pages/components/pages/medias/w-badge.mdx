# WBadge

WBadge @material-ui Badge'dan türetilmiştir.
Badge'un propertylerine erişmek için [tıklayın](https://material-ui.com/api/badge/).

### Ek Özellikler

Eklenen ve değiştirilen özellikler aşağıdaki gibidir:

| Adı                 | Veri Tipi                                                     | Default | Açıklama                                        |
|:--------------------|:--------------------------------------------------------------|:--------|:------------------------------------------------|
|                     |                                                               |         |                                                 |
