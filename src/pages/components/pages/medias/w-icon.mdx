# WIcon

WIcon @material-ui Icon'dan türetilmiştir.
Icon'un propertylerine erişmek için [tıklayın](https://material-ui.com/api/icon/).

### Ek Özellikler

Eklenen ve değiştirilen özellikler aşağıdaki gibidir:

| Adı                 | Veri Tipi                         | Default          | Açıklama                                         |
|:--------------------|:----------------------------------|:-----------------|:-------------------------------------------------|
| icon                | string                            |                  | Seçilen icon adı                                 |
| iconSize            | 'small' | 'default' | 'large'     | 'default'        | İkon boyutu                                      |
| iconSource          | 'material-icons' | 'fontawesome'  | 'material-icons' | İkon kütüphanesi                                 |

<br/>

> Material-icons Icon listesine [__Material Icons__](https://material.io/tools/icons/?style=baseline) adresinden erişebilirsiniz.
