# WAvatar

WAvatar @material-ui Avatar'dan türetilmiştir.
Avatar'un propertylerine erişmek için [tıklayın](https://material-ui.com/api/avatar/).

### Ek Özellikler

Eklenen ve değiştirilen özellikler aşağıdaki gibidir:

| Adı                 | Veri Tipi                                                     | Default | Açıklama                                        |
|:--------------------|:--------------------------------------------------------------|:--------|:------------------------------------------------|
|                     |                                                               |         |                                                 |


### Örnekler

<LiveCode title="Resim Avatarlar" code={`
<>
  <WFace.WAvatar 
    style={{margin: 10}}
    alt="bein HD1" 
    src="https://www.w3schools.com/howto/img_avatar.png"
  />
  <WFace.WAvatar 
    style={{margin: 10, width: 60, height: 60}}
    alt="bein HD1" 
    src="https://www.w3schools.com/howto/img_avatar.png"

</>
`}/>
