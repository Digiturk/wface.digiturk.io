# WLink

WLink @material-ui Link'dan türetilmiştir.
Link'un propertylerine erişmek için [tıklayın](https://material-ui.com/api/link/).

### Ek Özellikler

Eklenen ve değiştirilen özellikler aşağıdaki gibidir:

| Adı                 | Veri Tipi                                                     | Default | Açıklama                                        |
|:--------------------|:--------------------------------------------------------------|:--------|:------------------------------------------------|
|                     |                                                               |         |                                                 |
