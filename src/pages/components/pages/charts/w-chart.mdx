# WChart

WChart Recharts bileşenlerinden türetilmiştir.
Recharts ile ilgili dökümantasyona erişmek için [tıklayın](http://recharts.org/).

### Ek Özellikler

Eklenen ve değiştirilen özellikler aşağıdaki gibidir:

| Adı                 | Veri Tipi                                                     | Default | Açıklama                                        |
|:--------------------|:--------------------------------------------------------------|:--------|:------------------------------------------------|
|                     |                                                               |         |                                                 |
