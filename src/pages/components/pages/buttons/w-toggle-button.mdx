# WToggleButton

WToggleButton @material-ui ToggleButton'dan türetilmiştir.
ToggleButton'un propertylerine erişmek için [tıklayın](https://material-ui.com/api/toggle-button/).

### Ek Özellikler

Eklenen ve değiştirilen özellikler aşağıdaki gibidir:

| Adı                 | Veri Tipi                                                     | Default | Açıklama                                        |
|:--------------------|:--------------------------------------------------------------|:--------|:------------------------------------------------|
|                     |                                                               |         |                                                 |
