# WButtonGroup

WButtonGroup @material-ui ButtonGroup'dan türetilmiştir.
ButtonGroup'un propertylerine erişmek için [tıklayın](https://material-ui.com/api/button-group/).

### Ek Özellikler

Eklenen ve değiştirilen özellikler aşağıdaki gibidir:

| Adı                 | Veri Tipi                                                     | Default | Açıklama                                        |
|:--------------------|:--------------------------------------------------------------|:--------|:------------------------------------------------|
|                     |                                                               |         |                                                 |
