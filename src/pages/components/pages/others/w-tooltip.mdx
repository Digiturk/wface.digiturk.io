# WTooltip

WTooltip @material-ui Tooltip'den türetilmiştir.
Tooltip'un propertylerine erişmek için [tıklayın](https://material-ui.com/api/tooltip/).

### Ek Özellikler

Eklenen ve değiştirilen özellikler aşağıdaki gibidir:

| Adı                 | Veri Tipi                                                     | Default | Açıklama                                        |
|:--------------------|:--------------------------------------------------------------|:--------|:------------------------------------------------|
|                     |                                                               |         |                                                 |
