# WFace Framework

<MultiLang lang="en">
WFace Framework is a frontend development platform that based on ReactJs. 
It is designed for single page application development and contains many components. 
</MultiLang>
<MultiLang lang="tr">
WFace Framework, React tabanlı bir önyüz uygulama geliştirme platformudur. 
Tek sayfa uygulaması geliştirme maksatlı kullanım için tasarlanmıştır. İçerisinde bir çok bileşen barındırır.
</MultiLang>

<MultiLang lang="en">
It is independent backend technologies and communication styles. You can build your backend application with any 
technology. Generally REST services are used as communication style in WFace applications, though it is not mandatory.
Web socket or any other communication protocol that supported by browsers could be use.
</MultiLang>
<MultiLang lang="tr">
Backend teknolojisinden ve bağlantı yöntemlerinden tamamen bağımsız tasarlanmıştır. Backend uygulamanızı herhangi bir 
programlama dili ile geliştirebilir ve herhangi bir platformda çalıştırabilirsiniz. 
Backend servislerine erişimde genel olarak REST kullanılsa da bu bir zorunluluk değildir. 
Web socket veya tarayıcıların izin verdiği başka iletişim protokolleri de kullanılabilir.
</MultiLang>

## NodeJs

<MultiLang lang="en">
Nodejs is a runtime platform like JVM and .NET CLR.
It makes possible to run EcmaScript (Javascript) codes on machine. 
We will use npm that is package manager of Nodejs.
</MultiLang>
<MultiLang lang="tr">
Nodejs tıpkı JVM ve .NET CLR gibi bir runtime platformudur. 
EcmaScript (Javascript) kodlarımızın çalıştırılmasını sağlar. 
Biz NodeJs'in daha çok paket yönetim sistemi olan npm modülünü kullanacağız. 
</MultiLang>

<MultiLang lang="en">
> You can access npm commands and more detail by clicking [__here__](https://docs.npmjs.com/).
</MultiLang>
<MultiLang lang="tr">
> Npm ile ilgili detaylı bilgiye ve komutlara [__buradan__](https://docs.npmjs.com/) erişebilirsiniz.
</MultiLang>

## React 

<MultiLang lang="en">
React is component base UI library for web application development. 
You should be use React to develop a WFace application
</MultiLang>
<MultiLang lang="tr">
React, bileşen tabanlı uygulama geliştirmenize yarayan bir görsel kütüphanedir. 
WFace ile uygulama geliştirebilmek için React'i kullanabiliyor olmak gerekir. 
</MultiLang>

<MultiLang lang="en">
> If you are now familiar to React, firstly take a look at [__reactjs.org__] (https://reactjs.org/docs/getting-started.html).
</MultiLang>
<MultiLang lang="tr">
> Eğer React'a aşina değilseniz öncelikle [__reactjs.org__] (https://reactjs.org/docs/getting-started.html)
adresinden React'ı inceleyebilirsiniz.
</MultiLang>

## Typescript

<MultiLang lang="en">
Typescript is programming language that developed by Microsoft as open source. 
It adds some new features to Javascript. 
Typescript is being use in every phase of development of a WFace application.
All code are being converted to Javascript to run in browser in compile time.
</MultiLang>
<MultiLang lang="tr">
Typescript, Microsoft tarafından açık kaynak olarak geliştirilmiş ve 
Javascripte bazı özellikler katan yeni bir programlama dilidir. WFace platformunda tüm kodlamalar 
Typescript ile yapılmakta ve derleme zamanında Javascripte çevrilerek tarayıcı tarafından çalıştırılmaktadır. 
</MultiLang>

<MultiLang lang="en">
> To access more detail about Typescript click [__here__](https://www.typescriptlang.org/).
</MultiLang>
<MultiLang lang="tr">
> Typescript ile alakalı daha detaylı bilgiye [__buradan__](https://www.typescriptlang.org/) erişebilirsiniz.
</MultiLang>
