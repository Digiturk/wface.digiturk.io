## Zorunlu Değişiklikler

`wface.config.ts` dosyasında verilen configuration nesnesi içindeki `LoginScreen` değişkeni components.LoginPage şeklinde verilmelidir. 

```javascript
LoginScreen: MyLoginScreen, // Bu satır silinecek 

// Yerine bu şekilde tanımlama yapılacak. 
components:{
  LoginPage: MyLoginScreen 
}
```
